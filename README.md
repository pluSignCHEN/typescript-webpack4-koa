##  一个使用 typescript + webpack4 + nodejs 编写的koa程序框架

####  项目集成了nodemon热更新及webpack4打包依赖

需要执行:

  npm i

及

  npm i -g typescript webpack webpack-cli cross-env

现在，可以在本项目的基础上使用typescript编写您的koa程序

####  可以使用以下命令

启动程序及热更新：

  npm run dev

打包：

  npm run build
